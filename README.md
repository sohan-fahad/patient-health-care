## Our Services
DocTime is an online medical service app where patients can get 24/7 video consultation from doctors. DocTime uses 256 bit encryption to secure the video consultation. Patient can keep their previous consultation history and view online prescriptions.

Doctor can join the platform using our simple on boarding process. We verify every doctor to make sure only BMDC authorised doctors are providing consultation using latest technologies.
